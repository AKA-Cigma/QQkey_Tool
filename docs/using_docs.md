### smtp服务器配置
smtp服务器一般填完邮箱密码程序都会自动识别，但若识别失败，请手动查询对应smtp服务器输入
以下为常见邮箱的smtp服务器:

| 邮箱     | smtp服务器       |
|--------|---------------|
| 163邮箱  | smtp.163.com  |
| 126邮箱  | smtp.126.com  |
| QQ邮箱   | smtp.qq.com   |
| 139邮箱  | smtp.139.com  |
| yeah邮箱 | smtp.yeah.net |
